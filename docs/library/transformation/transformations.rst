:mod:`transformation.transformations` --- Transformations
=========================================================

The :mod:`transformation.transformations` module contains stateful
objects to support transformations between different coordinates spaces.
Currently, 3x3 homography and 3x3 fundamental matrices are supported
to convert between pairwise image spaces.
